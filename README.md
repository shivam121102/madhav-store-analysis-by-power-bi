# Madhav Store Sales Analysis – Power BI Dashboard

This project analyzes the e-commerce sales data of **Madhav Store** using Power BI.  
The goal of this project is to transform raw sales data into meaningful insights through interactive dashboards and visualizations.

## Project Objective

The main objective of this project is to help business stakeholders understand sales performance and customer behavior using data visualization.

The dashboard provides insights into:

- Sales trends
- Customer purchasing behavior
- Product category performance
- Regional sales distribution
- Profitability analysis

## Dataset Description

The dataset contains information such as:

- Order ID
- Order date
- Customer name
- State and city
- Product category
- Quantity
- Profit
- Payment mode

These datasets were used to analyze the business performance of the store.

## Analysis Performed

The following analysis was performed:

- Total sales and profit analysis
- Sales by state
- Category-wise sales performance
- Monthly profit trends
- Payment mode analysis
- Top customers analysis

## Dashboard Features

- KPI Cards (Total Sales, Profit, Quantity)
- Category-wise sales visualization
- State-wise performance analysis
- Interactive filters and slicers
- Monthly trend analysis

## Tools & Technologies

- Power BI
- Power Query
- Data Modeling
- DAX Measures
- Data Visualization

## Key Insights

Some important insights discovered from the analysis:

- Certain states contribute the highest revenue
- Clothing and electronics are major product categories
- Seasonal trends affect monthly sales
- Cash on Delivery is a widely used payment method
